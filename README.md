Timeboxer: Herramienta de Planificacion por Bloques
Timeboxer es un sistema de gestion de tiempo diseñado para organizar el dia de forma visual y estructurada. El proyecto implementa la tecnica de bloqueo de tiempo para asignar periodos especificos a tareas concretas, evitando la saturacion y mejorando el enfoque.

Caracteristicas del Diseño
Division de Tiempo: La agenda esta organizada en bloques de 30 minutos.
Espacios de Prioridad: Incluye secciones dedicadas para las tres prioridades principales y un area de descarga mental o notas libres.

Detalles Tecnicos
El desarrollo se ha realizado buscando la maxima ligereza y control sobre el codigo:
Frontend: Desarrollado en React para una gestion eficiente de los componentes y las rutas.
Estilos: Implementado con CSS puro
